Privacy Policy

Last updated: July 2026

1. Overview

This is a personal automation tool operated by a single developer for their own use. It is not a public product, and it does not collect, process, or store data belonging to any user other than its developer.

2. What data is used

The tool uses TikTok's Login Kit to authenticate the developer's own TikTok account, and the Content Posting API to publish video content to that same account. The only credentials stored are the developer's own OAuth access and refresh tokens, kept locally on the developer's own machine, and used solely to authenticate future requests to TikTok's API on the developer's behalf.

3. What data is not collected

This tool does not collect data from any other TikTok user or account, does not access follower lists, messages, or analytics of other accounts, and does not sell, share, or transmit any data to third parties.

4. Data retention

OAuth tokens are retained locally only for as long as needed to keep the developer's own automation running, and can be deleted at any time by removing the local token file.

5. Changes

This policy may be updated as the tool changes. Continued use after an update constitutes acceptance of the revised policy.
